# HRMS - Human Resource Management System

A comprehensive web application for managing candidates, onboarding, documents, and offer letters. Built with **React.js (Vite)** for the frontend, **Node.js + Express** for the backend, and **PostgreSQL** as the database. Styled using **TailwindCSS** with **shadcn/ui** components.

---

## Features

- Add, edit, and view candidates.
- Manage candidate information: name, email, phone, position, department, resume.
- Responsive UI with dark mode support.
- PostgreSQL database integration for persistent storage.
- Toast notifications for actions (success/failure).

---

## Tech Stack

- Frontend: React.js + Vite + TailwindCSS + shadcn/ui
- Backend: Node.js + Express
- Database: PostgreSQL
- API Client: Axios
- UI Components: Lucide icons, shadcn/ui Dialogs, Inputs, Selects
- Notifications: Sonner toast
